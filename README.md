<h1 style="text-align:center;">
  Mongogram
</h1>
<p style="text-align:center">Social media backend api using golang and mongodb</p>


## Features
- Authentication
    - Login
    - Signup
- Accounts
    - Change password
    - Deactivate account
    - Update profile
    - Reset password
- Media
    - upload media in gridfs
    - Fetch media from gridfs

- Search
    - Atlas search
    - Fetch recent searches
    - Clear specific recent search
    - Empty recent searches

- Post
    - Create new post
    - Fetch post lists
    - Fetch post details

- User
    - Follow user
    - Unfollow user
    - Fetch user lists
    - Fetch current user 
    - Fetch user profile
